Setup After iPad Reset

This guide walks through preparing and enrolling a Cricket iPad for store use, including Wi-Fi configuration, Ingenico pairing, app verification, and cloud printer setup.

üì± iPad Initial Setup
  1. On the Hello screen, press the Home button.
  2. Select English as the language.
  3. Select United States as the region.
  4. Appearance: Choose Default.
  5. If prompted, choose Set Up Without Another Device.
  6. Connect to CricketPOS Wi-Fi network.
  7. If prompted, continue with Partial Setup.
  8. At Remote Management, choose Enroll this iPad.
  9. Turn on Location Services.
  10. Tap Get Started.
