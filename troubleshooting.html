Interactive Troubleshooting Guide

Follow the steps below to diagnose and resolve pairing issues between an iPad and Ingenico device.

1️⃣ Check Bluetooth Icon on Ingenico

Look for the Bluetooth icon on the Ingenico display:

  • No icon → Bluetooth is disabled
  • White icon → Bluetooth on, not paired
  • Orange icon → Searching for iPad
  • Green icon → Connected successfully

2️⃣ No Bluetooth Icon? Enable Bluetooth

Enable Bluetooth on the Ingenico device:

  1. Press 0001 on the Cricket screen → Admin Menu
  2. Press 2 → Bluetooth Parameters
  3. Select 1 → “Switch On”
  4. If only “Switch Off” appears, toggle off → wait 10s → switch back on
  5. If still not working, reboot Ingenico (.,#* + Yellow keys)

3️⃣ Bluetooth Icon is Orange

Orange icon → paired but cannot locate iPad.

  • Ensure iPad Bluetooth is ON (Settings → Bluetooth)
  • Keep devices close together
  • Verify Ingenico is paired to correct iPad:
    1. Press 2634 → F key → Control Panel → Terminal Settings → Comm means → Bluetooth → Paired Devices
    2. Compare paired name to iPad’s Bluetooth name

4️⃣ Bluetooth Icon is Green

Green icon → connection established. Verify communication:

  1. Press 2634 → F key → Control Panel → Terminal Settings → Comm means → PCL
  2. If PCL shows “connected” → Aktivate communicates successfully
  3. If PCL shows “not connected” → restart Aktivate app on iPad

5️⃣ Still Not Working?

If issues persist after following all steps:

  • Attempt re-pairing the iPad & Ingenico (follow Exceed Article)
  • Contact Tier II Support
